# Hongik Community

홍익대학교 총동아리연합회 소개 사이트

## 프로젝트 개요

홍익대학교의 모든 연합동아리를 한눈에 소개하고, 동아리 활동을 효과적으로 홍보할 수 있는 통합 플랫폼입니다.

## 주요 기능

### 1. 사용자 관리
- **3단계 권한 시스템**
  - 웹사이트 총 관리자: 전체 시스템 관리
  - 동아리 관리자: 소속 동아리 정보 관리
  - 일반 사용자: 동아리 정보 조회

### 2. 동아리 소개
- 동아리 기본 정보 관리
  - 회장 정보
  - 연락처
  - 동아리방 위치
  - 모집 기간
  - 정규 모임 일정
- **마크다운 기반** 상세 소개 페이지

### 3. 활동 피드
- Instagram 계정 연동을 통한 자동 피드
- 직접 이미지 업로드 기능
- 동아리별 활동 갤러리

## 기술 스택

### Frontend
- React + TypeScript
- Vite
- Tailwind CSS + shadcn/ui
- React Router

### Backend (예정)
- Node.js + Express / NestJS
- PostgreSQL / MySQL
- Prisma ORM
